# 課金プラン設定の見直し提案

## 現状の問題点と改善案

### 1. 問題数制限の検討

#### 現状の設定
- **無料プラン**: 500問まで
- **プレミアムプラン**: 無制限
- **実際の問題数**: 1,500問以上（PROJECT_STATUS_REPORT.mdより）

#### 問題点
1. **500問は全体の約1/3**: 1,500問中500問は、ユーザーが満足する前に制限に達する可能性が高い
2. **学習体験の断絶**: 中途半端な段階で制限に達し、プレミアムへの誘導が強すぎる印象を与える
3. **競合比較**: 他の学習アプリでは無料プランでももっと多くの問題を提供している場合が多い

#### 改善案

**案1: 問題数を増やす（推奨）**
```
- 無料プラン: 1,000問まで（全体の約2/3）
- プレミアムプラン: 無制限 + 最新問題への優先アクセス
```

**案2: 時間ベースの制限**
```
- 無料プラン: 問題数制限なし、ただし1日10問まで
- プレミアムプラン: 無制限
```

**案3: 機能差別化**
```
- 無料プラン: すべての問題にアクセス可能、ただし：
  - AI解説は月10回まで
  - 模擬試験は1回まで/月
  - 広告表示あり
- プレミアムプラン: 全機能無制限 + 広告非表示
```

### 2. AI解説制限の検討

#### 現状の設定
- **無料プラン**: 月10回まで
- **プレミアムプラン**: 無制限

#### 問題点
1. **10回は少なすぎる**: 宅建試験は理解が重要なため、AI解説の制限が学習効果を大きく下げる
2. **転換率への影響**: 制限が厳しすぎると、ユーザーが離脱する可能性が高い

#### 改善案

**推奨案**
```
- 無料プラン: 月30回まで（1日1回程度）
- プレミアムプラン: 無制限
```

**代替案**
```
- 無料プラン: 週10回まで（毎日1-2回程度）
- プレミアムプラン: 無制限
```

### 3. 問題のクオリティについて

#### 現状
- AI生成問題が多数含まれている
- 過去問ベースの問題は著作権の問題で削除されている

#### 懸念点
1. **AI生成問題の品質**: 問題の正確性や難易度のバランス
2. **過去問との整合性**: 実際の試験問題との類似性
3. **ユーザーからの信頼性**: AI生成問題に対するユーザーの信頼度

#### 改善案

1. **品質保証プロセス**
   - 専門家による問題レビュー
   - ベータテストによる品質確認
   - ユーザーフィードバックの収集と改善

2. **問題の明確化**
   - 問題の出典を明記（AI生成問題であることを明示）
   - 過去問ベースの問題とAI生成問題を区別して表示
   - 問題の信頼度スコアを表示

3. **段階的なリリース**
   - 高品質な問題から優先的にリリース
   - ユーザーフィードバックに基づいて問題を改善

### 4. プラン設定の最終推奨案

#### 無料プラン（推奨設定）
```
✅ 問題演習: 1,000問まで（または機能制限なし、時間制限あり）
✅ AI解説: 月30回まで
✅ 過去問: 直近3年分
✅ 基本的な進捗管理
✅ 弱点分析（基本版）
❌ 広告表示あり
❌ 模擬試験: 月1回まで
❌ 高度な分析機能: なし
❌ オフライン問題: なし
```

#### プレミアムプラン（推奨設定）
```
✅ 問題演習: 無制限
✅ AI解説: 無制限
✅ 過去問: 10年分以上
✅ すべての進捗管理機能
✅ 弱点分析（詳細版）
✅ 模擬試験: 無制限
✅ 合格者パターン分析
✅ スペーシング復習機能
✅ 広告非表示
✅ オフライン問題: 無制限
✅ 優先サポート
✅ 最新問題への優先アクセス
```

### 5. 価格設定の見直し

#### 現状
- 月額: 980円
- 年額: 9,800円（約17%割引）

#### 改善提案

**案1: 価格を下げる**
```
- 月額: 680円
- 年額: 6,800円（約17%割引）
理由: より多くのユーザーに手が届く価格帯に
```

**案2: 現状維持**
```
- 月額: 980円
- 年額: 9,800円
理由: 機能が充実していれば適正価格
```

**案3: 段階的プラン**
```
- ライトプラン: 月額480円（AI解説無制限のみ）
- スタンダードプラン: 月額980円（現プレミアムプラン）
- プレミアムプラン: 月額1,480円（全機能 + 個別指導）
```
