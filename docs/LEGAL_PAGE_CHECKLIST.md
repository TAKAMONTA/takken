## ⚠️ 要対応項目

### 🔴 高優先度：事業者情報の入力

以下の情報を `/app/legal/page.tsx` に入力してください：

#### 1. 販売事業者名

- [ ] 現在: `【要入力：正式な事業者名または個人事業主名】`
- [ ] 対応: 正式な事業者名を入力

**入力例：**

- 法人の場合: `株式会社〇〇〇` / `合同会社〇〇〇`
- 個人事業主: `個人事業主 山田太郎`

#### 2. 運営統括責任者

- [ ] 現在: `【要入力：責任者名】`
- [ ] 対応: 責任者の氏名を入力

**入力例：**

- `山田 太郎`

#### 3. 所在地

- [ ] 現在: `【要入力：郵便番号・住所】`
- [ ] 対応: 事業所の住所を入力

**入力例：**

- `〒100-0001 東京都千代田区千代田1-1-1 〇〇ビル3F`
- `〒530-0001 大阪府大阪市北区梅田1-1-1`

**注意事項：**

- 自宅住所を公開したくない場合は、バーチャルオフィスや私書箱の利用を検討
- 請求があれば遅滞なく開示する旨を記載（既に記載済み）

#### 4. 電話番号

- [ ] 現在: `【要入力：連絡可能な電話番号】`
- [ ] 対応: 連絡可能な電話番号を入力

**入力例：**

- `03-1234-5678`
- `050-1234-5678`（IP 電話可）
- `090-1234-5678`（携帯電話可）

**注意事項：**

- メール対応を優先したい旨は既に記載済み
- 050 番号や IP 電話の利用も可能
