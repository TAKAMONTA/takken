## データ形式

各問題データには以下の情報が含まれています：
- 問題文（完全な文章）
- 4つの選択肢（完全な文章）
- 正解番号
- 詳細な解説
- カテゴリ分類
- 難易度設定
- 年度情報

## データ追加の要件

新しい年度のデータを追加する際は、以下の情報が必要です：
1. 問題文（完全な文章）
2. 4つの選択肢（それぞれ完全な文章）
3. 正解番号
4. 解説文（学習効果を高めるため）

正解番号のみでは学習アプリとしての機能を十分に提供できないため、
完全なデータセットが揃ってから追加することを推奨します。
