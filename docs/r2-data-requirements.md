# 令和2年度試験データの追加について

## 現状の課題

提供されたPDFファイルには以下の情報のみが含まれています：
- 令和2年10月試験の正解番号（問1-50）
- 令和2年12月試験の正解番号（問1-50）
- 合否判定基準

## 既存システムのデータ構造

現在のシステムでは、各問題に以下の情報が必要です：

```typescript
interface Question {
  id: number;           // 問題ID
  question: string;     // 問題文（完全な文章）
  options: string[];    // 選択肢（4つの完全な文章）
  correctAnswer: number;// 正解番号（0-3のインデックス）
  explanation: string;  // 解説文
  category: string;     // カテゴリ（minpou, hourei, zeihou, takkengyouhou）
  difficulty: string;   // 難易度
  year: string;        // 年度
}
```

## 必要な追加情報

令和2年度のデータをシステムに追加するには、以下の情報が必要です：

### 1. 民法等（問1-14）
- 各問題の完全な問題文
- 各問題の4つの選択肢の完全な文章
- 各問題の解説

### 2. 法令上の制限（問15-22）
- 各問題の完全な問題文
- 各問題の4つの選択肢の完全な文章
- 各問題の解説

### 3. 税法（問23-25）
- 各問題の完全な問題文
- 各問題の4つの選択肢の完全な文章
- 各問題の解説

### 4. 宅建業法（問26-50）
- 各問題の完全な問題文
- 各問題の4つの選択肢の完全な文章
- 各問題の解説

## 対応方法の提案

### オプション1：完全なデータの取得
- 令和2年度の問題集や過去問題集から、問題文と選択肢を入手
- 各問題に対する解説を作成または入手

### オプション2：段階的な実装
1. まず正解番号のみを保存する簡易版を作成
2. 後日、問題文や選択肢を追加していく
3. ただし、この場合は既存のクイズ機能では使用できない

### オプション3：外部リソースの活用
- 宅建試験の公式サイトや信頼できる教育機関から問題文を取得
- APIやデータベースサービスを利用して問題データを取得

## 推奨される次のステップ

1. 令和2年度の完全な問題文と選択肢を含む資料を入手
2. 入手した資料を基に、既存のデータ構造に合わせてデータを作成
3. 各カテゴリごとにファイルを作成（r2-10.ts, r2-12.ts）
4. インデックスファイルを更新して新しいデータを統合

## 注意事項

- 正解番号のみでは、学習アプリとしての機能を十分に提供できません
- ユーザーが問題を解く際には、問題文と選択肢が必要不可欠です
- 解説がないと、学習効果が大幅に低下します
