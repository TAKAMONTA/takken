### ⚠️ 問題点・要修正項目

1. **Firebase認証エラー**
   - 新規登録時に400エラーが発生
   - エラーメッセージ: `Registration error: JSHandle@error`
   - 原因: Firebase認証の設定またはAPIキーの問題の可能性

2. **PWAアイコン404エラー**
   - `/icons/icon-16x16.png` が見つからない
   - `/icons/icon-32x32.png` が見つからない
   - `/icons/icon-144x144.png` が見つからない
   - 影響: PWAとしてインストール時にアイコンが表示されない

3. **メタタグの非推奨警告**
   - `apple-mobile-web-app-capable` メタタグが非推奨
   - 推奨: `mobile-web-app-capable` メタタグを追加

## 推奨される次のアクション

1. **Firebase設定の確認**
   - `.env.local`のFirebase設定値を確認
   - Firebase Consoleで認証設定を確認
   - メール/パスワード認証が有効になっているか確認

2. **PWAアイコンの追加**
   - `public/icons/`ディレクトリにアイコンファイルを配置
   - 各サイズのアイコンを用意（16x16, 32x32, 144x144など）

3. **メタタグの更新**
   - `app/layout.tsx`でメタタグを更新

4. **Androidビルドテスト**
   ```bash
   npm run build
   npx cap sync android
   npx cap open android
   ```
